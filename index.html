Developer Skills and Learning Pathways: An Evidence-Based Framework for Strategic Human Resource Developmentexpand_more

Preprint You, H. S. (2025)

This study presents a comprehensive, interpretable machine learning analysis of the 2024 Stack Overflow Developer Survey...

Full Abstractclose

This study presents a comprehensive, interpretable machine learning analysis of the 2024 Stack Overflow Developer Survey to investigate the market value of developer skills and the effectiveness of various learning pathways. Employing XGBoost models and SHAP value interpretation, we estimate skill premiums, value impacts, and learning method ROI across developer subgroups. Results reveal optimal learning approaches for different career stages and technical domains, providing a data-driven framework for strategic human resource development.
